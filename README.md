## 📖 Sobre o Projeto

O **Sistema de Venda de Ingressos - Rock in Rio 2026** é uma aplicação web desenvolvida para simular a comercialização de ingressos para eventos musicais.

O projeto utiliza **HTML5**, **CSS3**, **JavaScript** e **Bootstrap 5**, proporcionando uma experiência moderna, responsiva e intuitiva para o usuário.

---

## 🚀 Funcionalidades

- 🎟️ Seleção de ingressos através de cards interativos
- 💰 Cálculo automático do valor total da compra
- 📋 Resumo dinâmico da compra
- 🌙 Alternância entre modo claro e escuro
- 💾 Persistência do tema utilizando LocalStorage
- ✔️ Validação de formulário
- 📱 Layout responsivo
- 🎠 Carrossel de destaques do evento
- ❓ FAQ utilizando Accordion
- 🔔 Popovers informativos
- ✨ Animações ao rolar a página
- 🧭 Menu com destaque automático da seção ativa

## 🌙 Sistema de Temas

O usuário pode alternar entre:

- ☀️ Modo Claro
- 🌙 Modo Escuro

A preferência é salva automaticamente no navegador através do **LocalStorage**.

---

## ⚙️ Recursos JavaScript

### 🎯 Seleção Inteligente

Ao clicar em um card de ingresso:

- O ingresso é selecionado automaticamente.
- O formulário é atualizado.
- O card recebe destaque visual.
- A página realiza scroll suave até o formulário.

### 💰 Resumo da Compra

O sistema exibe em tempo real:

- Tipo do ingresso
- Quantidade selecionada
- Valor total

### ✔️ Validação

O formulário valida:

- Nome completo
- E-mail
- Tipo de ingresso
- Quantidade permitida

### 🎉 Confirmação da Compra

Após o envio:

```text
Matheus, sua compra foi registrada:
2 ingresso(s) para Área VIP.
Total: R$ 500,00
```
